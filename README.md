# Building 3D Model

[![Render OpenSCAD Preview](https://github.com/atommych/openscad-house/actions/workflows/render.yml/badge.svg)](https://github.com/atommych/openscad-house/actions/workflows/render.yml)

Parametric OpenSCAD 3D model of a two-storey mixed-use building in Portugal.

> Renders are auto-generated by CI on every push — see the [`renders/`](renders/) folder.

---

## File Structure

| File | Description |
|------|-------------|
| `building.scad` | Main model — geometry, modules, assembly |
| `variables.scad` | All parametric values (edit here to tune the model) |
| `.github/workflows/render.yml` | CI: auto-renders PNG + STL on every push |

---

## Modules

| Module | Description |
|--------|-------------|
| `ground_floor()` | RC: Café Snack-Bar (50 m²), Sala de Jantar (27 m²), Bar + Copa, WC block |
| `first_floor()` | 1st floor: Sala (4.5×4.3), Cozinha (4.0×3.4), 3× Quarto, Casa de Banho |
| `garage()` | Separate garage (5.5×4.5) with Terraço slab above |
| `staircase()` | Straight stair flight connecting RC → Andar |
| `wall_with_opening()` | Utility: wall box with door/window cut-out |
| `room_label()` | Utility: 3D extruded text label |

---

## Key Parameters (`variables.scad`)

```openscad
wall_thickness = 0.2;    // [m] wall thickness
wall_height    = 2.8;    // [m] floor-to-ceiling height
floor_slab     = 0.15;   // [m] concrete slab thickness

door_w  = 0.9;           // [m] interior door width
door_h  = 2.1;           // [m] interior door height
window_w = 1.2;          // [m] window width
window_h = 1.2;          // [m] window height
window_sill = 0.9;       // [m] sill height from floor

stair_steps = 16;        // number of treads
stair_rise  = 0.175;     // [m] riser height
stair_run   = 0.27;      // [m] tread depth
```

**Scale: 1 OpenSCAD unit = 1 metre**

---

## Usage

1. Install [OpenSCAD](https://openscad.org/) v2021.01 or later.
2. Clone this repo.
3. Open `building.scad` — parameters are loaded from `variables.scad` automatically.
4. **F5** — fast preview &nbsp;|&nbsp; **F6** — full render &nbsp;|&nbsp; **F7** — export STL
